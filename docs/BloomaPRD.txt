# Blooma 제품 요구사항 문서 (PRD)

## 1. 개요
Blooma는 패션/의류 브랜드를 위한 AI 이미지 생성 플랫폼입니다.
모델 사진과 의류 이미지를 조합하여 전문적인 룩북 이미지를 생성합니다.

**핵심 가치**
- 패션 브랜드가 포토슈팅 없이 룩북 이미지 생성
- 빠르고 비용 효율적인 제품 비주얼 제작
- 고품질 결과물로 실제 촬영 수준의 이미지

**핵심 메시지**
"Weekly lookbooks, no photoshoots."

## 2. 타겟 사용자
- 소규모 패션 브랜드 및 디자이너
- 마케팅 에이전시와 콘텐츠 크리에이터
- 이커머스 판매자

## 3. 목표
### 비즈니스 목표
- 패션 촬영 비용/시간 80% 절감
- 주간 단위 룩북/캠페인 이미지 생성 지원
- B2B 고객(에이전시/스튜디오) 구독 확보

### 사용자 목표
- 간편한 고품질 룩북 이미지 생성
- 다양한 모델/의상 조합 테스트
- 갤러리에서 생성 이미지 관리

## 4. 핵심 기능

### 4.1 Studio (이미지 생성)
- 모델 선택 (라이브러리 또는 업로드)
- 의류(Outfit) 참조 이미지 추가 (최대 5장)
- View 선택 (Front/Behind/Side/Quarter)
- 디테일 프롬프트 입력
- Standard/Pro 품질 모드 선택
- 해상도 선택 (Pro 모드: 2K/4K)
- 생성 개수 선택 (2장/4장)

### 4.2 Gallery
- 생성된 이미지 그리드 뷰
- 즐겨찾기 필터링
- 상세 모달 (소스 모델/의상 이미지 확인)
- 원본 다운로드

### 4.3 Assets (모델/로케이션 관리)
- 커스텀 모델 업로드 및 관리
- 시스템 제공 Public 모델
- 삭제 기능 (본인 업로드 에셋만)

### 4.4 크레딧 시스템
- 월간 크레딧 충전 (플랜별 차등)
- Standard: 10 크레딧/장
- Pro: 50 크레딧/장

### 4.5 구독/결제
- 3개 플랜: Small Brands / Agency / Studio
- 월간/연간 결제 옵션 (연간 20% 할인)
- Polar 결제 연동

## 5. 기술 요구사항
- 이미지 생성 응답 시간: 60초 이내
- UI 반응 속도: 200ms 이내
- 인증: Clerk
- 데이터 저장: Cloudflare D1 / R2

## 6. 로드맵

### Phase 1 (현재)
- Studio 이미지 생성
- Gallery 관리
- 구독/결제 시스템
- 모델 라이브러리

### Phase 2
- 커스텀 카메라 프리셋
- 배치 생성 및 스케줄링
- 팀 협업 기능

### Phase 3
- API 제공 (외부 연동)
- 화이트 라벨 솔루션
