# 제품 요구사항 문서 (PRD)

## 1. 개요
Blooma는 크리에이터를 위한 AI 기반 스토리보드 생성 플랫폼이다. 
아이디어를 입력하면 AI가 자동으로 시각적 스토리보드를 생성하여 콘텐츠 기획 시간을 70% 단축한다.

**핵심 가치:**
- 아이디어 → 스토리보드를 몇 분 만에 완성
- 브랜드 일관성을 유지하는 AI 이미지/비디오 생성
- 캐릭터 일관성 관리로 시리즈 콘텐츠 제작 지원

**제품 태그라인:**
"Your AI Studio is here."

## 2. 문제 정의
- 기획 경험이 부족한 크리에이터는 아이디어를 시각화하기 어려움
- 기존 AI 이미지 생성 도구는 브랜드/캐릭터 일관성 유지가 어려움
- 스토리보드 수동 작성 시 시간·노력이 과다 투입
- 콘텐츠 기획 전용 AI 솔루션의 부재

## 3. 목표 및 목적

### 주요 목표
- AI 자동 스토리보드 기능으로 기획 시간 70% 단축
- 브랜드/캐릭터 일관성을 유지하는 이미지 생성

### 보조 목표
- 기획 결과물의 시각적 품질 향상
- 스토리보드에서 비디오 클립으로 확장
- 팀·클라이언트와 공유 및 피드백 간소화

### 성공 지표
- MAU 10,000명
- 주간 재방문률 40% 이상
- AI 생성 스토리보드 완료율 80%
- 평균 페이지 로드 ≤ 1.5초

## 4. 대상 사용자

### 1차 사용자
- 20~40대 1인·소규모 크리에이터 (유튜버, 틱톡커, 블로거)
- 콘텐츠 기획 경험이 낮거나 시간 제약이 큰 사용자
- 마케팅 에셋을 빠르게 제작해야 하는 개인/팀

### 2차 이해관계자
- 광고주·브랜드 (콘텐츠 발주 측)
- MCN·에이전시 기획팀
- 인디 게임/앱 개발자 (프로모션 콘텐츠 제작)

## 5. 사용자 스토리

### 스토리보드 생성
- "크리에이터로서, 스크립트를 입력하면 AI가 씬별 스토리보드를 자동 생성해주길 원한다."
- "마케터로서, 제품 컨셉만 입력하면 마케팅 에셋 시안을 빠르게 받고 싶다."

### 캐릭터 일관성
- "콘텐츠 제작자로서, 같은 캐릭터가 모든 씬에서 일관되게 등장하길 원한다."
- "시리즈 크리에이터로서, 등록한 캐릭터를 다른 프로젝트에서도 재사용하고 싶다."

### 비디오 확장
- "유튜버로서, 완성된 스토리보드 이미지를 바로 짧은 비디오 클립으로 변환하고 싶다."

### 협업
- "협업 파트너로서, 웹 링크로 스토리보드를 열람하고 피드백을 주고 싶다."

## 6. 기능 요구사항

### 핵심 기능 (구현 완료)

#### 1. AI 스토리보드 생성
- 스크립트 입력 → AI가 씬 분할 및 프롬프트 자동 생성
- 카드 기반 시퀀스 시각화 (Hook, Problem, Solution, Evidence, Benefit, CTA)
- 드래그&드롭으로 카드 순서 변경
- 카드 추가, 삭제, 복제 기능

#### 2. AI 이미지 생성
- 다중 AI 모델 지원:
  - Seedream v4 (Text-to-Image, Edit)
  - Nano Banana Pro (Text-to-Image, Edit)
  - Nano Banana (경량 모델)
- 다양한 화면 비율: 16:9, 9:16, 1:1, 4:3, 3:4, 3:2, 2:3
- 해상도 옵션: 1K, 2K, 4K
- 참조 이미지 기반 편집 (최대 3장)

#### 3. AI 비디오 생성
- Image-to-Video 변환 (Kling v2.5 Turbo Pro/Standard)
- Start-End Frame 비디오 (2장 이미지 → 모션 비디오)
- 5초/10초 영상 길이 선택

#### 4. 캐릭터 관리
- AI 캐릭터 생성 위저드
- 캐릭터 이미지 업로드
- 프롬프트에 @멘션으로 캐릭터 참조
- 프로젝트 간 캐릭터 재사용

#### 5. 비주얼 에셋 라이브러리
- 카메라 프리셋 (앵글, 샷 타입)
- 배경 라이브러리
- 모델(3D/인물) 라이브러리

#### 6. 크레딧 시스템
- 기능별 크레딧 차등 소비:
  - Seedream v4: 10 크레딧
  - Nano Banana Pro: 50 크레딧
  - Kling Video Pro: 120 크레딧
  - Kling Video Standard: 70 크레딧
- Polar 결제 연동
- 월간 구독 플랜:
  - Starter: $19/월, 2,200 크레딧
  - Pro: $49/월, 6,000 크레딧
  - Studio: $99/월, 13,000 크레딧

#### 7. 내보내기
- PNG 개별 다운로드
- PDF 스토리보드 내보내기
- 비디오 클립 다운로드 (MP4)

### 부가 기능 (개발 예정)
- AI 아이디어 제안 (대화형 기획 지원)
- 실시간 협업 및 코멘트
- Google Drive 연동
- 템플릿 마켓플레이스
- 다국어 지원 (한국어, 영어)

## 7. 비기능 요구사항

### 성능
- 페이지 초기 로드: ≤ 1.5초
- 이미지 생성 요청: ≤ 30초
- UI 반응 시간: < 200ms

### 보안
- Clerk OAuth 2.0 인증
- Cloudflare D1 데이터베이스 보안
- R2 스토리지 서명된 URL

### 사용성
- 반응형 디자인 (데스크톱/모바일)
- 다크 모드 지원
- WCAG 2.1 AA 접근성 준수

### 확장성
- 월 100만 API 호출 대응 가능 구조
- Vercel 오토스케일링

### 호환성
- Chrome, Safari, Edge 최신 2버전 지원

## 9. 리스크 및 대응

### 기술 리스크
- AI 생성 품질 편차 → 다중 모델 폴백, 사용자 재생성 옵션
- 외부 API 장애 → 재시도 로직, 플레이스홀더 이미지 제공

### 비즈니스 리스크
- API 호출 비용 증가 → 크레딧 기반 과금으로 비용 전가
- 저작권 이슈 → 안전 필터 적용, 가이드라인 제공
